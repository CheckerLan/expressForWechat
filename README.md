# 有个快递

## 项目介绍

在微信小程序平台开发,使用云开放技术,利用快递鸟api,对个人快递信息查询管理

## 项目目录

- **cloudfunctions** 云函数文件夹
- **miniprogram** 小程序主程序
  - **pages** 页面文件夹
  - **components** 组件文件夹
  - **images** 图片文件夹
  - **style** 样式文件夹
  - app.js 公用js
  - app.json 公用配置
  - app.wxss 公用样式
  - sitemap.json sitemap配置文件
- proje.config.json 项目配置
